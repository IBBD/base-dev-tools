# Linux基础开发工具集合

镜像地址：https://hub.docker.com/r/ibbd/base-dev-tools/

## 安装列表

- man
- python2.7
- git, git-flow
- vim（spf13-vim插件）
- tmux 
- mycli: mysql命令行美化工具
